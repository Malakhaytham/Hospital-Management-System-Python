# 🏥 Hospital Management System

A desktop Hospital Management System developed using Python, Object-Oriented Programming (OOP), Tkinter GUI, and SQLite.

## Features
- Add Doctors
- Add Patients
- Add Staff
- Search Patient
- Create New Visit
- View Doctor's Patients
- Store Data using SQLite Database

## Technologies Used
- Python
- OOP
- Tkinter
- SQLite
